# Cơ Học Đất - Giáo trình trực tuyến

Ứng dụng giáo trình Cơ Học Đất bằng tiếng Việt, xây dựng bằng React.

## Tính năng
- 📚 11 chương đầy đủ dịch từ sách gốc
- 🌙 Dark / Light mode
- 🔍 Tìm kiếm bài học
- 🎯 Kiểm tra kiến thức cuối chương
- 📱 Tối ưu mobile

## Tech stack
- React 18
- Tailwind-style CSS-in-JS
- Vercel deployment
